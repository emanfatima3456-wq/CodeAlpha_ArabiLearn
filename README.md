# 🌙 ArabiLearn - Arabic Learning App

A beautiful and interactive Arabic language learning app built with Flutter.

---

## ✨ Features

- 🏠 **Home Dashboard** — Progress overview and quick access to all features
- 📚 **Vocabulary Screen** — Learn Arabic words with categories and search
- 🃏 **Flashcard Screen** — 3D flip cards for interactive learning
- 🧠 **Quiz Screen** — Multiple choice quiz with animations and score tracking
- 📈 **Progress Screen** — Real-time progress tracking with achievements
- 🤖 **AI Arabic Tutor** — Powered by Cohere AI for instant Arabic help
- 🌙 **Dark Theme** — Professional and eye-friendly dark UI

---

## 🛠️ Tech Stack

- **Flutter** — Cross-platform mobile development
- **Dart** — Programming language
- **Cohere AI API** — AI-powered Arabic tutor
- **SharedPreferences** — Local data persistence
- **animate_do** — Beautiful animations

## 🚀 Getting Started

### Prerequisites
- Flutter SDK
- Android Studio
- Cohere API Key

### Installation

```bash
git clone https://github.com/emanfatima3456-wq/CodeAlpha_ArabiLearn.git
cd CodeAlpha_ArabiLearn
flutter pub get
flutter run
```

### API Setup
1. Go to [dashboard.cohere.com](https://dashboard.cohere.com)
2. Create a free account
3. Generate an API key
4. Add it to `lib/utils/constants.dart`

## 🏆 Achievements System

| Achievement | Requirement |
|-------------|-------------|
| 🔥 Week Streak | 7 day streak |
| 📚 Word Master | 20 correct answers |
| 🧠 Quiz Pro | Complete 3 quizzes |
| ⭐ XP Hunter | Earn 100 XP |
| 🏅 Champion | Complete 10 quizzes |
| 💎 Legend | Earn 500 XP |
